Mini projet integration 

 Fonctionnalités

- En-tête moderne :
  - Logo et navigation (Home, Shop, Magazine)
  - Barre de recherche et menu déroulant de tri
  - Bouton de connexion et panier

- Barre latérale de filtres :
  - Filtrage par collection, couleur, catégorie
  - Filtrage par plage de prix (input range)

- Grille de produits :
  - Affichage des produits sous forme de cartes
  - Chaque carte contient : image, nom, catégorie, prix, note et bouton panier
  - Organisation dynamique en 3 colonnes sur desktop

- Pagination :
  - Liens de pages numérotées en bas de page

- Responsive :
  - Adaptation automatique sur tablette et mobile (1 à 2 colonnes)
  - Sidebar devient horizontale sur petits écrans


 Technologies utilisées

- HTML5 : structure de la page
- CSS3 (Flexbox et Grid) : style et mise en page responsive
